    Stock Analysis using Page Hits

    For this mini project I wanted to see if I could analyse the performance of a stock using page hits to Wikipedia. This is a technique that I have seen used by a few others on the internet, but I wanted to see if I could do it from scratch using JavaScript. Wikipedia has a REST API which allows access to page views. I wanted to see if these correlated with the stock price of differnet companies.
